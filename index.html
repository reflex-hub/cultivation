                    /**
                     * sort Used to order an array of integers in ascending order (O(n)).
                     * @param {Array} arr
                     * @return {Promise}
                     */
                    
                    function sort (arr) {
                        return new Promise(resolve => {
                            const sorted = [];
                            
                            for (const item of arr) {
                                setTimeout(item => sorted.push(item), item, item);
                            }
                            
                            setTimeout(() => resolve(sorted), arr.reduce((a, b) => a + b, 0) + 1);
                        });
                    }

For example, the two vision-related points GB37 (gall bladder) and UB60 (urinary bladder) showed deactivation in visual brain areas like the Cuneus. The team concluded that Acupuncture seems to affect the brain’s processing of both physical Sensations and Emotions.
